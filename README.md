# Socketry

High-level wrappers for Ruby sockets with advanced thread-safe timeout support.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'socketry'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install socketry
